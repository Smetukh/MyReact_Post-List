# MyReact_Post-List
Created with CodeSandbox

Create new posts with this application. 
You can change post state, filter, search and other.

Application's tech stack:
 - ReactJS
 - Redux
 - recompose
 - reselect
 - react-router-dom
 - redux-thunk

Live demonstration is at: 
https://react-post-list.netlify.app
